This tool uses the Web MIDI API, which runs in Chrome or Edge on desktop (not Safari or Firefox). To get going:

  • Allow MIDI access if prompted.
  • If ports are missing, check chrome://flags and your OS MIDI/audio permissions.

VT-4 note semantics: notes sent set the harmonic context (key/scale center), not the pitch the VT-4 outputs. Sending C–E–G tells it to harmonize around C major. VT-4 MIDI channel defaults to 1; change above if your VT-4 listens elsewhere.
Signal chain: condenser mic → VT-4 (phantom ON, rear XLR) → L OUT (wet) + R OUT (dry/bypass) → 707 EXT IN L/R. Set 707 EXT IN type = LINE. The R/BYPASS jack carries the dry vocal — route it to one side of the looper to capture dry while monitoring wet.
