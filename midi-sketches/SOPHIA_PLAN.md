# Sophia Beds — the plan (repeatable)

Backing beds for **impromptu singing**: long, learnable, layerable, with the
vocal register left empty. Generated by `python sophia.py` (seeded — the same
beds come back every time).

## Why each design choice

| Choice | Reason |
|---|---|
| ~2–3 min per bed | long enough to develop an idea without restarting |
| intro → A → lift → A → lift → tail | a dynamic arc *invites* phrasing: verse energy, chorus lift, closing breath |
| one 4-chord cycle per family, repeating every 4 bars | a singer learns it in one pass, then improvises with confidence |
| nearest-inversion pad voice leading | chords glide instead of jump — nothing yanks the ear |
| pads voiced **below C4**, arps **above C5**, bass in the basement | the octave where the voice lives (~C4–C5) has zero melodic competition |
| `drone.mid` root-anchor stem | warm-up reference; can sit under any subset |
| timeline-aligned stems | any subset dropped into the same slots lines up (bass enters after the intro, arp at the first lift — by design) |
| 707-native drums (pads 36–51) | straight onto the groovebox |
| groove-engine feel per family | every stem in a family shares one pocket |

## The six families

| Bed | Cycle | Character |
|---|---|---|
| 01_Ember_72_Am | i–VI–III–VII | warm Rhodes, head-nod — the all-purpose one |
| 02_Vesper_64_Ddor | i–IV vamp | organ, ceremonial toms, hypnotic two-chord dwell |
| 03_Sable_80_Gm | lament descent | strings, halftime — for the melancholy ones |
| 04_Aurora_88_Cmaj | I–V–vi–IV | warm pad, soft four-floor — open, hopeful |
| 05_Nocturne_68_Fsm | i–VII–VI–VII | choir, very spare — maximal room for the voice |
| 06_Solstice_96_Dmaj | I–V–vi–IV | e-piano, driving — the energetic one |

## How to use (MC-707 or DAW)

1. Start with `bed_full.mid` in a family whose mood fits.
2. Or build up: `drone.mid` alone to find the key → add `pad.mid` → add
   `drums.mid` + `bass.mid` when she wants a groove → `arp.mid` for lift.
3. Stems only mix **within** a family (same key/tempo/feel).

## How to repeat / extend

- Regenerate: `python sophia.py` (identical output — seeded).
- New bed = **one line** in `FAMILIES`:
  `(seed, name, bpm, root_midi, scale, chord_cycle_degrees, feel, pad_prog, arp_prog, bass_prog)`
  Pick a chord cycle from the proven set (Axis `[0,4,5,3]`, aeolian pop
  `[0,5,2,6]`, lament `[0,-1,-2,-3]`, vamp `[0,3,0,3]`) or invent one — degrees
  are scale-degrees of the family's mode.
- If her comfortable range isn't ~C4–C5, shift `REG_PAD`/`REG_ARP` and the keys.
- Invariants guarded by `python tests.py`; the pack is registered in
  `make_everything.py`, so every future everything-zip includes it.
