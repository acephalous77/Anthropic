Setup, once: connect the 707 by USB → on the box, set each track's MIDI Rx channel (SHIFT + TRACK SEL) to match the channels here, and set Control Rx = ON in the MIDI tab so scene Program Changes land. Pick the output port above. In the box's Matrix Control, assign a CC to any partial parameter, then type that CC number into a track's slot here.
Honest limits of v1: this is the control layer only. It sends notes, Program Change, transport, and CC — no patch editing (that needs the box itself or the B67 editor), and Scatter/mute aren't pre-wired because their MIDI mapping isn't documented; map them yourself via Matrix Control and use a CC slot. Web MIDI runs in Chrome or Edge (not Safari/Firefox), so open this file there.
